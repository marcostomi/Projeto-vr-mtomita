Museu de Torres Antigas - Projeto VR
Descrição
Este projeto foi desenvolvido como parte da trilha de aprendizado em Web 3.0 e Metaverso. O objetivo é proporcionar uma experiência imersiva em Realidade Virtual (VR) que apresenta a evolução das torres de comunicação ao longo da história recente da humanidade.

A aplicação consiste em uma galeria virtual contendo três mostruários distintos. Cada um exibe um modelo de torre específico, com poltronas posicionadas estrategicamente para permitir a contemplação detalhada e o estudo das estruturas.

🛠️ Tecnologias Utilizadas
Engine: Unity 6 (6000.0.13f1 LTS)

SDK de VR: Meta XR All-in-One SDK (OVRInteraction, Building Blocks)

Plataforma Alvo: Android (Meta Quest)

🚀 Funcionalidades e Aprendizados
Integração de Building Blocks: Utilização do Camera Rig e OVRInteractionComprehensive para configurar rapidamente o ambiente de Realidade Virtual.

Cenografia e Iluminação: Criação de um ambiente de galeria com materiais personalizados (conforme visível na pasta Assets/Materials na imagem) para simular texturas de vidro, tecidos e metais.

Organização de Hierarquia: Estruturação de objetos complexos e otimização de modelos 3D para renderização em dispositivos móveis.

🧠 Desafios Superados
Durante o desenvolvimento, foram enfrentados desafios técnicos relacionados à configuração do ambiente no Unity e resolução de erros de compatibilidade com o Project Setup Tool. A superação desses obstáculos permitiu uma compreensão mais profunda do pipeline de desenvolvimento para o Meta Quest.
